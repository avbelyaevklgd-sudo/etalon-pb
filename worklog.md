**1. Хедер (Header):**

- Копирайт

- Card hover: scale(1.05) + box-shadow при наведении

- Glassmorphism эффект для карточек на фоновом изображении
- Плавные переходы (transitions) для всех интерактивных элементов
- Shimmer-эффект для белых карточек
- Кастомные scrollbar стили

**5. Адаптивность:**
- Мобильное меню (бургер) для экранов < 1280px
